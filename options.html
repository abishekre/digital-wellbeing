Screen Timer Options

Sites to not track
(one per line) 		Screen Timer will not track data of any website on this list. Existing data for these sites will also be removed.
Sites to Restrict Usage
(one per line) 		Screen Timer will restrict usage of websites on this list. The corresponding time entered will be the maximum time allowed.
Chart Display Limit Sites below the limit will be grouped into the "Other" category.
Storage There is no way to recover your data once you delete it.
